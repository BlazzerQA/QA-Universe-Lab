Java Collections: Тренажер

1. ArrayList vs LinkedList: в чем разница? ▾
Ответ: ArrayList — массив (быстрый доступ), LinkedList — список (быстрая вставка в начало).

2. Как работает HashMap? ▾
Ответ: Использует hashCode() для поиска корзины и equals() для поиска внутри корзины (связный список/дерево).

3. TreeSet vs HashSet? ▾
Ответ: HashSet не гарантирует порядок (O(1)), TreeSet хранит элементы в отсортированном виде (O(log n)).
